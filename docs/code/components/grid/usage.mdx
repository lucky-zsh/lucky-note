# Button组件用法

## 基础按钮

该组件不包含具体的样式，具体的需要根据自己的需要进行设置

```tsx
import { KButton } from '@/ui/index';

const KButtonDemo1 = function () {
    return <KButton>按钮</KButton>;
};

export default KButtonDemo1;
```

## 禁用按钮

该组件不包含具体的样式，具体的需要根据自己的需要进行设置

```tsx
import { KButton } from '@/ui/index';

const KButtonDemo2 = function () {
    return <KButton disabled={true}>按钮</KButton>;
};

export default KButtonDemo2;
```

## 文字按钮

该组件不包含具体的样式，具体的需要根据自己的需要进行设置

```tsx
import { KButton } from '@/ui/index';

const KButtonDemo3 = function () {
    return (
        <div>
            <KButton text={true}>按钮</KButton>
            <KButton text={true} disabled={true}>
                按钮
            </KButton>
        </div>
    );
};

export default KButtonDemo3;
```

## 链接按钮

该组件不包含具体的样式，具体的需要根据自己的需要进行设置

```tsx
import { KButton } from '@/ui/index';

const KButtonDemo4 = function () {
    return (

```



## Tag 按钮

可以使用 button ，div ，a 标签作为最外城的包裹元素

```tsx
import { KButton } from '@/ui/index';

const KButtonDemo5 = function () {
    return <KButton>在这里可以任意定制按钮的内容</KButton>;
};

export default KButtonDemo5;
```

## 自定义内容按钮

只提供外层的基本样式，内部内同可以自由的去定制和扩展

```tsx
import { KButton } from '@/ui/index';

const KButtonDemo6 = function () {
    return <KButton>在这里可以任意定制按钮的内容</KButton>;
};

export default KButtonDemo6;
```
