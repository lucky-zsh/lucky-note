# CSS Module

## 介绍

CSS Modules 是一个流行的，用于模块化和组合 CSS 的系统，它是通过编译的方式修改选择器名字为全局唯一的方式来实现 CSS 的样式隔离。如果对一个小项目来说这个可能不是必要的，但如果涉及大型团队项目，使用CSS Module 恰恰是一个很好的选择，
接下来我们就一起来学习一下如何使用更好Css Module。

## 原理

我们使用CSS Module 可以在Webpack中的css-loader中配置，那今天我们就看看CSS-loader的源码来看看CSS-Module是如何实现的，首先打开[css-loader 源码地址](https://github.com/webpack-contrib/css-loader/blob/master/src/utils.js),
我们看到又这四个模块的引入,如下代码，这个就是CSS-Module实现所必须的库

```ts
import modulesValues from 'postcss-modules-values';
import localByDefault from 'postcss-modules-local-by-default';
import extractImports from 'postcss-modules-extract-imports';
import modulesScope from 'postcss-modules-scope';
```
